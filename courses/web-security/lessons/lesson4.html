SQL Injection Basics

What is SQL Injection?

SQL Injection is a type of attack that allows malicious users to execute arbitrary SQL commands on a database. This can lead to unauthorized access, data theft, or data manipulation.

Common SQL Injection Techniques

Types of SQL Injection:

  • Union-Based: Using UNION operator to combine results
  • Error-Based: Extracting data through error messages
  • Boolean-Based: Using TRUE/FALSE conditions
  • Time-Based: Using time delays to extract data

Vulnerable Code Examples


// Vulnerable code
const query = `SELECT * FROM users WHERE username = '${username}' AND password = '${password}'`;
db.query(query);  // Dangerous!

// Secure code using prepared statements
const query = 'SELECT * FROM users WHERE username = ? AND password = ?';
db.query(query, [username, password]);  // Safe!

Prevention Techniques

Best Practices:

  • Use Prepared Statements
  • Input Validation
  • Escape Special Characters
  • Principle of Least Privilege
